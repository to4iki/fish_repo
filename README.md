# fish_repo

Select ghq repository or get

## Install

With [fisherman]  
With [peco]  
With [ghq]  

```
fisher to4iki/fish_repo
```

## Usage

Please bind repo to your favorite key.  
If you'd like to bind CTRL+]

```fish
function fish_user_key_bindings
  bind \c] repo
end
```
